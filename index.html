HTML - A Estrutura da Web

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Define a estrutura do conteúdo usando tags.

Estrutura Básica

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
</head>
<body>
    <!-- Conteúdo aqui -->
</body>
</html>

Cabeçalhos e Parágrafos

<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<p>Este é um parágrafo de texto. <br>Isto é uma quebra de linha.</p>
<p>Este é outro parágrafo com texto em <b>negrito</b> e <i>itálico</i>.</p>

Listas Ordenadas e Não Ordenadas

<h4>Lista de Compras:</h4>
<ul>
    <li>Leite</li>
    <li>Pão</li>
    <li>Ovos</li>
</ul>

<h4>Passos:</h4>
<ol>
    <li>Primeiro passo</li>
    <li>Segundo passo</li>
</ol>

<!-- Vídeo -->

    <source src="video.mp4" type="video/mp4">

CSS - Estilizando a Web

CSS (Cascading Style Sheets) define estilos visuais para HTML. Controla cores, fontes, espaçamentos e layout.

Seletores e Cores

/* Estiliza o corpo da página */
body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}

/* Estiliza todos os h1 */
h1 {
    color: navy;
    text-align: center;
}

/* Estiliza elementos com a classe 'destaque' */
.destaque {
    background-color: yellow;
}

Modelo de Caixa (Box Model)

div {
    width: 300px;
    padding: 20px; /* Espaçamento interno */
    border: 5px solid gray; /* Borda */
    margin: 15px; /* Espaçamento externo */
}

Centralização e Alinhamento

/* Centralizar horizontalmente */
.centro {
    display: block;
    margin: 0 auto;
}

/* Alinhar à esquerda */
.esquerda {
    float: left;
}

/* Alinhar à direita */
.direita {
    float: right;
}

/* Posicionar no topo */
.topo {
    position: absolute;
    top: 0;
}

/* Posicionar na parte inferior */
.base {
    position: absolute;
    bottom: 0;
}

Alinhamento de Texto

/* Centralizar texto */
.texto-centro {
    text-align: center;
}

/* Alinhar texto à esquerda */
.texto-esquerda {
    text-align: left;
}

/* Alinhar texto à direita */
.texto-direita {
    text-align: right;
}

/* Justificar texto */
.texto-justificado {
    text-align: justify;
}

Alinhamento de Imagem ou Vídeo

/* Centralizar imagem ou vídeo */
.midia-centro {
    display: block;
    margin: 0 auto;
}

/* Alinhar à esquerda */
.midia-esquerda {
    float: left;
    margin-right: 10px; /* Espaço ao lado */
}

/* Alinhar à direita */
.midia-direita {
    float: right;
    margin-left: 10px; /* Espaço ao lado */
}

/* Centralizar usando flex */
.container-centro {
    display: flex;
    justify-content: center;
    align-items: center;
}

Posicionamento de Div em Qualquer Lugar

/* Posição fixa na tela */
.div-fixa {
    position: fixed;
    top: 20px;
    right: 20px;
}

/* Posição absoluta dentro do elemento pai */
.div-absoluta {
    position: absolute;
    bottom: 10px;
    left: 10px;
}

/* Centralizar horizontal e vertical com position */
.div-centro {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}

/* Usando flex para centralizar dentro do pai */
.pai-flex {
    display: flex;
    justify-content: center;
    align-items: center;
}

Estilo e Posição de Botão

/* Botão básico */

    background-color: #007BFF; /* Azul */
    color: white; /* Cor do texto */

    background-color: #0056b3; /* Azul mais escuro ao passar o mouse */

/* Centralizar o botão */
.botao-centro {
    display: block;
    margin: 0 auto;
}

/* Alinhar à direita */
.botao-direita {
    float: right;
}

/* Alinhar à esquerda */
.botao-esquerda {
    float: left;
}

CSS: Cabeçalho e Rodapé (Fixo ou Acompanhando Scroll)

/* ====== Estilos Gerais ====== */
body {
    margin: 0;
    font-family: Arial, sans-serif;
}

/* ====== Cabeçalho ====== */
header {
    background-color: #004080; /* Cor de fundo do cabeçalho */
    color: white; /* Cor do texto */
    padding: 15px 20px; /* Espaçamento interno */
}

/* MENU DENTRO DO CABEÇALHO */
header nav ul {
    list-style: none; /* Remove marcadores da lista */
    margin: 0;
    padding: 0;
    display: flex; /* Deixa os itens lado a lado */
    justify-content: center; /* Centraliza os links no menu */
}

header nav ul li {
    margin: 0 15px; /* Espaçamento entre os itens */
}

header nav ul li a {
    color: white; /* Cor dos links */
    text-decoration: none; /* Remove sublinhado */
    font-weight: bold;
    transition: color 0.3s;
}

header nav ul li a:hover {
    color: #ffcc00; /* Cor ao passar o mouse */
}

/* ====== Opção para Cabeçalho FIXO ====== */
/* Adicione esta classe ao 
 para deixá-lo fixo */
header.fixo {
    position: fixed; /* Mantém o cabeçalho visível mesmo com rolagem */
    top: 0;
    left: 0;
    width: 100%;
    z-index: 1000; /* Fica acima do conteúdo */
}

/* ====== Rodapé ====== */
footer {
    background-color: #333; /* Cor de fundo do rodapé */
    color: white; /* Cor do texto */
    text-align: center; /* Centraliza o texto */
    padding: 15px 20px; /* Espaçamento interno */
}

/* ====== Opção para Rodapé FIXO ====== */
/* Adicione esta classe ao 
 para deixá-lo fixo */
footer.fixo {
    position: fixed; /* Mantém o rodapé visível sempre */
    bottom: 0;
    left: 0;
    width: 100%;
    z-index: 1000;
}

/* ====== Conteúdo com Cabeçalho ou Rodapé fixo ====== */
/* Evita que o conteúdo fique escondido atrás do cabeçalho fixo */
main {
    padding: 80px 20px; /* Espaço quando o cabeçalho for fixo */
}

JavaScript - A Interatividade da Web

JavaScript permite adicionar interatividade e lógica às páginas web, manipulando o DOM, validando formulários e muito mais.

Manipulação do DOM

// Encontra um elemento pelo seu ID
const titulo = document.getElementById('titulo');

// Altera o texto do elemento
if (titulo) {
    titulo.textContent = 'Olá, Mundo do JS!';
    titulo.style.color = 'blue';
}

Funções e Lógica

function verificarIdade(idade) {
    if (idade >= 18) {
        return 'Maior de idade.';
    } else {
        return 'Menor de idade.';
    }
}

console.log(verificarIdade(25)); // Exibe no console do navegador

PHP - Scripting do Lado do Servidor

PHP é uma linguagem de script interpretada no servidor, amplamente usada para criar páginas web dinâmicas e sistemas completos.

Variáveis e Saída (echo)

<?php
$saudacao = "Olá, Mundo!";
$ano = 2025;

echo $saudacao;
echo "<br>"; // Gera uma quebra de linha em HTML
echo "Estamos no ano de " . $ano;
?>

Lógica Condicional (if/else)

<?php
$temperatura = 15;

if ($temperatura > 25) {
    echo "Está calor.";
} else {
    echo "Não está calor, pode levar um casaco.";
}
?>

Arrays e Laços (foreach)

<?php
$frutas = ["Maçã", "Banana", "Laranja"];

echo "<ul>";
foreach ($frutas as $fruta) {
    echo "<li>" . $fruta . "</li>";
}
echo "</ul>";
?>

Java - Robusta e Multiplataforma

Java é orientada a objetos, multiplataforma e utilizada para aplicações desktop, web, Android e sistemas corporativos.

Estrutura Básica e Saída

public class MeuPrograma {
    public static void main(String[] args) {
        String mensagem = "Olá, mundo Java!";
        System.out.println(mensagem);
    }
}

Variáveis e Tipos de Dados

int idade = 30;
double preco = 49.99;
char inicial = 'J';
boolean ativo = true;

System.out.println("Idade: " + idade);

Laços de Repetição (for)

public class Tabuada {
    public static void main(String[] args) {
        int numero = 7;
        for (int i = 1; i <= 10; i++) {
            System.out.println(numero + " x " + i + " = " + (numero * i));
        }
    }
}

Python - Simples e Poderosa

Python é conhecida por sua sintaxe simples e clareza, sendo aplicada em análise de dados, inteligência artificial, automação e web.

Variáveis e Função print()

# Python é dinamicamente tipado
nome = "Maria"
idade = 28
altura = 1.65

print(f"Nome: {nome}")
print(f"Idade: {idade} anos")
print(f"Altura: {altura}m")

Estruturas Condicionais

nota = 7.5

if nota >= 7.0:
    print("Aprovado!")
elif nota >= 5.0:
    print("Recuperação.")
else:
    print("Reprovado.")

Listas e Laços (for)

# Uma lista de strings
planetas = ["Mercúrio", "Vênus", "Terra", "Marte"]

for planeta in planetas:
    print(f"Visitando o planeta {planeta}...")
    
print("Fim da viagem!")
